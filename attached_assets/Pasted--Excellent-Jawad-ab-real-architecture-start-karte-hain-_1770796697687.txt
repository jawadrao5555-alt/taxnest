# 🚀 PHASE 2 — Company Isolation Middleware

---

## 🟢 STEP 1 — Middleware Code Add Karo

Open file:

```
app/Http/Middleware/CompanyIsolation.php
```

Replace poora file content with this:

```php
<?php

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;
use Symfony\Component\HttpFoundation\Response;

class CompanyIsolation
{
    public function handle(Request $request, Closure $next): Response
    {
        if (auth()->check()) {
            $companyId = auth()->user()->company_id;

            if (!$companyId) {
                abort(403, 'No company assigned.');
            }

            // Share company_id globally
            app()->instance('currentCompanyId', $companyId);
        }

        return $next($request);
    }
}
```

Save karo.

## 🟢 STEP 4 — Middleware Register Karo

Open:

```
bootstrap/app.php
```

Find middleware section (Laravel 11 style).

Add this inside:

```php
->withMiddleware(function ($middleware) {
    $middleware->alias([
        'company' => \App\Http\Middleware\CompanyIsolation::class,
    ]);
})
```
