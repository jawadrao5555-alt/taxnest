# 🚀 PHASE 7 — Role Based Permission (Proper Enterprise Version)

Ab hum serious RBAC (Role-Based Access Control) implement karte hain.

---

## 🟢 STEP 1 — Users Table Mein Role Column Add

Terminal:

```bash
php artisan make:migration add_role_to_users_table
```

Migration file open karo aur paste:

```php
public function up(): void
{
    Schema::table('users', function (Blueprint $table) {
        $table->string('role')->default('employee')->after('email');
    });
}
```

Run:

```bash
php artisan migrate
```

---

## 🟢 STEP 2 — User Model Update

Open:

`app/Models/User.php`

Add:

```php
protected $fillable = [
    'name',
    'email',
    'password',
    'company_id',
    'role'
];

public function isSuperAdmin()
{
    return $this->role === 'super_admin';
}

public function isCompanyAdmin()
{
    return $this->role === 'company_admin';
}

public function isEmployee()
{
    return $this->role === 'employee';
}

public function isViewer()
{
    return $this->role === 'viewer';
}
```

---

## 🟢 STEP 3 — Role Middleware

```bash
php artisan make:middleware RoleMiddleware
```

Open:

`app/Http/Middleware/RoleMiddleware.php`

Replace content:

```php
public function handle($request, Closure $next, ...$roles)
{
    if (!auth()->check()) {
        abort(403);
    }

    if (!in_array(auth()->user()->role, $roles)) {
        abort(403, 'Unauthorized role.');
    }

    return $next($request);
}
```

---

## 🟢 STEP 4 — Middleware Register

`bootstrap/app.php` mein add karo:

```php
$middleware->alias([
    'role' => \App\Http\Middleware\RoleMiddleware::class,
]);
```

---

## 🟢 STEP 5 — Protect Routes

Example:

```php
Route::middleware(['auth', 'company', 'role:super_admin'])->group(function () {
    Route::get('/admin', function () {
        return "Super Admin Panel";
    });
});

Route::middleware(['auth', 'company', 'role:company_admin,employee'])->group(function () {
    Route::get('/invoice/create', [InvoiceController::class, 'create']);
});
```
