# 🚀 PHASE 12 — Visual Analytics (Chart.js Integration)

Ab hum dashboard ko data-driven banayenge.

---

## 🟢 STEP 1 — Chart.js Install

Terminal:

```bash
npm install chart.js
npm run build
```

---

## 🟢 STEP 2 — Company Dashboard Controller Upgrade

Open dashboard route logic (web.php mein jo closure hai)
Usko controller-based bana dete hain.

Terminal:

```bash
php artisan make:controller DashboardController
```

Open:

`app/Http/Controllers/DashboardController.php`

Replace with:

```php
<?php

namespace App\Http\Controllers;

use App\Models\Invoice;

class DashboardController extends Controller
{
    public function index()
    {
        $companyId = app('currentCompanyId');

        $invoices = Invoice::where('company_id', $companyId)->get();

        $monthlyData = Invoice::where('company_id', $companyId)
            ->selectRaw('COUNT(*) as count, MONTH(created_at) as month')
            ->groupBy('month')
            ->pluck('count', 'month');

        return view('dashboard', compact('invoices', 'monthlyData'));
    }
}
```

---

## 🟢 STEP 3 — Route Update

`routes/web.php`:

Replace dashboard route:

```php
use App\Http\Controllers\DashboardController;

Route::middleware(['auth', 'company'])->group(function () {
    Route::get('/dashboard', [DashboardController::class, 'index']);
});
```

---

## 🟢 STEP 4 — Add Chart to Dashboard View

Open:

`resources/views/dashboard.blade.php`

Add below KPI cards:

```blade
<canvas id="invoiceChart" class="bg-white p-4 rounded shadow"></canvas>

<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

<script>
const ctx = document.getElementById('invoiceChart');

new Chart(ctx, {
    type: 'bar',
    data: {
        labels: {!! json_encode(array_keys($monthlyData->toArray())) !!},
        datasets: [{
            label: 'Invoices Per Month',
            data: {!! json_encode(array_values($monthlyData->toArray())) !!},
            borderWidth: 1
        }]
    },
});
</script>
```
