# 🚀 PHASE 8 — Admin Analytics Dashboard

Ab hum ek real SaaS control panel banate hain.

---

## 🟢 STEP 1 — Admin Controller

Terminal:

```bash
php artisan make:controller AdminController
```

Open:

`app/Http/Controllers/AdminController.php`

Replace with:

```php
<?php

namespace App\Http\Controllers;

use App\Models\Company;
use App\Models\Invoice;
use App\Models\FbrLog;
use App\Models\User;

class AdminController extends Controller
{
    public function dashboard()
    {
        $totalCompanies = Company::count();
        $totalUsers = User::count();
        $totalInvoices = Invoice::count();
        $lockedInvoices = Invoice::where('status', 'locked')->count();
        $failedLogs = FbrLog::where('status', 'failed')->count();

        return view('admin.dashboard', compact(
            'totalCompanies',
            'totalUsers',
            'totalInvoices',
            'lockedInvoices',
            'failedLogs'
        ));
    }
}
```

---

## 🟢 STEP 2 — Route Protection

`routes/web.php` mein add karo:

```php
use App\Http\Controllers\AdminController;

Route::middleware(['auth', 'role:super_admin'])->group(function () {
    Route::get('/admin/dashboard', [AdminController::class, 'dashboard']);
});
```

---

## 🟢 STEP 3 — Admin View Create

Create folder:

```
resources/views/admin
```

Create file:

```
dashboard.blade.php
```

Paste:

```html
<h2>Super Admin Dashboard</h2>

<hr>

<p>Total Companies: {{ $totalCompanies }}</p>
<p>Total Users: {{ $totalUsers }}</p>
<p>Total Invoices: {{ $totalInvoices }}</p>
<p>Locked Invoices: {{ $lockedInvoices }}</p>
<p>Failed FBR Logs: {{ $failedLogs }}</p>
```
