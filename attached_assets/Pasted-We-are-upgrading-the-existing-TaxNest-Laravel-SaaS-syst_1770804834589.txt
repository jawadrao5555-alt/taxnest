PHASE 1 – Invoice Hardening
==============================

1. Add integrity_hash (SHA256) column to invoices table.
2. When invoice status becomes "locked", generate SHA256 hash from:
   invoice_number + total_amount + tax_amount + company_id + created_at
3. Store hash permanently.
4. Create invoice_activity_logs table:
   - id
   - invoice_id
   - company_id
   - user_id
   - action (created, edited, submitted, retry, locked)
   - changes_json (nullable)
   - ip_address
   - created_at
5. Log every invoice creation, edit, submission, retry, and lock.
6. Add invoice timeline component in dashboard view showing activity logs chronologically.
7. Add "Verify Integrity" button on invoice detail page that recalculates and compares SHA256.
